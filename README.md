# mirage-bootvar-solo5 -- Solo5 implementation of MirageOS Bootvar interface

Library for passing boot parameters from Solo5 to MirageOS.

## Install

Bootvar can be installed with `opam`:

```
opam install mirage-bootvar-solo5
```
